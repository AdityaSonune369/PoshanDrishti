# Requirements Document

## Functional Requirements
- Dual scan modes (Kitchen Plate & Health Check - palm/eyes)
- Real-time food ingredient detection
- Visible deficiency detection (anemia pallor, vitamin deficiency)
- Personalised 7-day meal plan generation
- Natural voice output in Hindi-English mix
- PDF export of plan

## Non-Functional Requirements
- 100% Offline operation
- Privacy-first (no data leaves device)
- Runs on standard student laptops
- Response time < 8 seconds
- Beautiful, intuitive UI
- Supports Indian languages & cultural context
