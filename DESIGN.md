# Design Document

## High-Level Design
- **Frontend**: Gradio with custom CSS animations
- **Vision Layer**: BLIP + OpenCV
- **Intelligence Layer**: Phi-3-mini-4k-instruct (4-bit)
- **Delivery Layer**: pyttsx3 + ReportLab

## User Flow
1. Launch → Choose mode
2. Scan → AI Analysis
3. Generate → Voice + Visual Plan
4. Export/Share

## Tech Stack
- Python, Gradio, Transformers, OpenCV, pyttsx3
- Optimised using AMD Ryzen AI Software Stack 1.7
